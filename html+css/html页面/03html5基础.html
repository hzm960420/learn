3.能说出常用的浏览器
4.能说出web标准的三大组成部分

1.网站是指在因特网上根据一定的规则，使用html等制作的用于展示特定内容相关的网页集合，通常是HTML格式的文件。网页是构成网站的基本元素，通常由图片、链接、文字、声音、视频等元素组成。

2.html指的是超文本标记语言，是用来描述网页的一种语言，超文本有两层意义：（1.它可以加入图片、声音、动画、多媒体等内容，超出了文本限制）（2.它可以从一个文件跳转到另一个文件，与世界各地主机的文件连接(超级链接文本)）

3.ie浏览器 火狐浏览器 谷歌浏览器 Safari Opera 五大浏览器（浏览器内核：负责读取网页内容，整理讯息，计算网页的显示方式并显示页面） (五大内核：Trident(ie)、Gecko(火狐)、Webkit(Safari)、chromium(chrome)、)

4.web标准的三大组成部分：结构(structure)、表现(persentation)、行为(behavior) 结构就是html 表现就是css 行为就是js
