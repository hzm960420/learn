使用表单目的是为了手机用户信息
一个完整的表单通常由表单域、表单控件(也称为表单元素)和提示信息3个部分构成
表单域是一个包含表单元素的区域，form元素标签用于定义表单域，以实现用户信息的收集和传递
各种表单元素控件
男 女
睡觉 吃饭 打豆豆
action: 用于指定接收并处理表单数据的服务器程序的url地址 method:用于设置表单数据提交方式 name:用于指定表单名称,以区分同一个页面的多个表单域
input输入表单
type属性:button checkbox file hidden image password radio reset submit text
input标签的属性： name: 定义input元素的名称 value: 规定input元素的值 checked: 规定此input元素首次加载时应当被选中 maxlength:规定输入字段中的字符的最大长度
