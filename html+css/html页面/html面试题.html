label标签是input元素定义标注标签，作用是点击label内容 能让绑定在label上的input元素起作用 当用户选择该标签时，浏览器会自动将焦点转到和标签相关的表单控件上

label有两种写法 一种是用for 另一种是嵌套

使用XHTML 1.XHTML中所有的标记都必须有一个相应的结束标签 2.XHTML所有标签的元素和属性都必须使用小写 3.所有标记都必须合理嵌套 4.所有的属性都必须有双引号包裹起来 5.把所有的"《">和‘&’用编码标识 6.图片必须加注释说明
