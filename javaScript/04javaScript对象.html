javaScript创建对象的方式： 1.利用字面量创建对象 2.利用new Object创建对象 3.利用构造函数创建对象
new关键字再执行时会做四件事: 1.在内存中创建一个新的空对象。 2.让this指向这个新的对象 3.执行构造函数里的代码,给这个对象添加属性和方法 4.返回这个新对象；
for....in语句用于对对象或数组进行循环操作
javaScript内置对象 1.Math Math.PI // 圆周率 Math.floor() // 向下取整 Math.ceil() // 向上取整 Math.around() //四舍五入 Math.abs() // 取绝对值 Math.max() / Math.min() // 求最大值和最小值 2.Date() var now = new Date(); Date()里不写参数 返回的是当前时间 Date()里写参数 返回的是参数里的时间 var date = newDate(); date.getFullYear(); // 获取当年 date.getMonth() + 1; //获取当月 范围0~11 date.getDate(); // 获取当天日期 date.getDay(); //获取星期几 date.getHours(); // 获取当前小时 date.getMinutes(); // 获取当前分钟 date.getSeconds(); // 获取当前秒 3.Arrray arr.push; // 数组末尾添加一个或多个数 arr.pop; // 数组末尾删除一个数 arr.shift(); // 数组开头删除一个或多个数 arr.unshift(); // 数组开头添加一个或多个数
检查是否为数组：instanceof isArray();
数组排序：reverse() sort() 数组索引：indexOf() lastIndexOf()
数组转换为字符串 toString() join()
数组的其他方法：concat() slice() splice()
