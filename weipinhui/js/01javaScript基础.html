javaScript是运行在客户端的脚本语言 Script是脚本的意思 脚本语言:不需要编译，运行过程中由js引擎逐行来进行解释并执行

浏览器本身不会执行js代码，而是通过内置javaScript引擎(解释器)来执行js代码。

js由：js语法 文档对象 浏览器对象组成 （ECMAScript、dom、bom）

1.行内式 js 2.内嵌式 3.外部引入

注释：单行注释 多行注释

javaScript的数据类型为 基本数据类型和引用数据类型 基本数据类型：string number boolean null undefined(5种) 引用数据类型 object array function

转换为字符串类型 Number.toString(); 强制转换String(num); 转换为数字型 parseInt parseFloat Number() 隐式转换 - * / console.log('12' - 0); // 12 console.log('123' * 1); //123 console.log('123' - '120'); 转换为boolean Boolearn();

逻辑与 如果第一个表达式为正确 那么返回表达式2 如果第一个表达式错误 那么返回表达式1 逻辑或 如果第一个表达式为正确 那么返回表达式1 如果第一个表达式错误 那么返回表达式2
