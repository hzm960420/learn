arguments: javaScript中当前函数的内置对象。arguments对象中存储了传递的所有实参。 1.具有length属性 2.按索引方式存储数组 3.不具有数组的push,pop等方法
函数的两种声明方式: 1.命名函数 2.匿名函数 function test1(){ } 调用函数的代码既可以放到声明函数的前面，也可以放在声明函数的后面 var fn = function(){ } 函数调用的代码必须写到函数体后面
javaScript作用域 1.全局作用域 script标签下的作用域 2.函数作用域 函数或方法体里面的范围 3.块级作用域(es6新增) 用let声明 全局作用域：在任何一个地方都可以使用，只有在浏览器关闭时才会被销毁，因此比较占内存 局部作用域：在函数调用时初始化,还函数执行完毕后销毁 不占内存
javaScript预解析：js引擎会把js里面所有的var 还有function提升到当前作用域的最前面
